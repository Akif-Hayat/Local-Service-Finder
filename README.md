# Local Service Finder

A small Flutter app that lists local service providers (electrician, plumber, carpenter, builder, etc.) and lets users call them. NOTE: regular users cannot add or edit providers from the app â€” the provider list is read-only.

## Quick features
- List of providers (name, category, phone, optional address/notes)
- Search or filter by category
- Tap to open the phone dialer and call a provider
